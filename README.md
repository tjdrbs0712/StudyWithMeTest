## 🔍 STUDY WITH ME 에서는...
- 게시글을 통해 함께 공부할 수 있는 사람을 구할 수 있어요
- 함께 하고 싶은 스터디가 있다면 댓글을 통해 스터디 참여 의사를 밝혀보세요

## 📌 핵심 기능
### 회원
- 회원 가입
- 회원 탈퇴
- 로그인
- 로그아웃
- 이메일 인증
- 타회원 정보 조회
- 내 정보 조회
- 내 정보 수정
- 비밀번호 변경

### 게시글
- 게시글 등록
- 게시글 단일 조회
- 게시글 전체 조회
  - 최신순
  - 좋아요순
  - 기간내 조회
- 게시글 수정
- 게시글 삭제
- 게시글 좋아요 등록 / 삭제

### 댓글
- 댓글 등록
- 댓글 조회
- 댓글 수정
- 댓글 삭제
- 댓글 좋아요 등록 / 삭제
